What is HTML?

HTML, expanded as Hyper Text Markup Language is a standard
markup language used in the creation of web pages. HTML combined
with CSS (Cascading Style Sheet) and JavaScript takes web development
to an advanced level. The formatting tags image, video, anchor, and hyperlink
tags help structure the web page in a variety of ways.
